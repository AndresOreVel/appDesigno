# Designo

![Vista previa de Designo](./preview.jpg)

Designo es un proyecto personal de desarrollo web que simula la página corporativa de una empresa con presencia en distintas ubicaciones del mundo.
El proyecto destaca por la integración de Leaflet para la visualización de las sedes de la empresa mediante un mapa interactivo.

La aplicación está organizada en tres secciones principales:
- About Us: presenta la información de la empresa y los servicios que ofrece,
incluyendo diseño web, diesño de aplicaciones y diseño gráfico.
- Locations: muestra las diferentes ubicaciones de la empresa a nivel global a través
de un mapa interactivo.
- Contact: incluye un formulario de contacto funcional para la comunicación con los usuarios.

## Tecnologías utilizadas

- Angular
- TypeScript
- Scss
- Leaflet
